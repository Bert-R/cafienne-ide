### Install pre-requesites

* install nodejs >= 4.2.2 (see [nodejs.org](http://nodejs.org)) and npm (comes with nodejs)
* install webpack

### Build and run

* run `npm install`

* run `webpack`

* run `node bin/www`

* open http://localhost:3317 in your browser
